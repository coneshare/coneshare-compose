# Coneshare Docker

This directory contains the Docker Compose setup to run **Coneshare** on your own infrastructure. Coneshare is an enterprise-grade, self-hosted document sharing and virtual data room solution.

For detailed application documentation, please refer to the main [Coneshare repository](https://github.com/coneshare/coneshare).

## Installation

Run the installation script from within this directory:
```bash
./install.sh
```
This will check for requirements, create configuration files, and pull the necessary Docker images.

After installation, start the application with:
```bash
./start.sh
```
